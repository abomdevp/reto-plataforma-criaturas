# reto-plataforma-criaturas-miticas

Sistema CRUD desarrollado con Vue 3 para gestionar criaturas míticas rescatadas, aplicando reactividad y Composition API.

# Funcionalidades

- Registro, edición y eliminación de criaturas (CRUD completo)
- Cálculo automático del Índice de Riesgo Total mediante computed
- Validaciones con modificadores v-model (.trim, .number, .lazy)
- Reporte de criaturas en cuarentena desde consola
- Visualización condicional de alertas y estado vacío

# Uso (si requiere o no instalar dependencias)

Requiere instalar dependencias con `npm install` y ejecutar con `npm run dev`.
